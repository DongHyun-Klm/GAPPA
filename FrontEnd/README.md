### 🔑Commit Convention 
|태그|설명|
|:--:|:--:|
|Feat|새로운 기능 추가|
|Fix|버그 수정|
|Docs|문서 수정|
|Refactor|코드 리펙토링|
|Test|테스트 코드|
|Chore|패키지 매니저 수정, 그외 기타 수정 ex).gitignore|
|Design|CSS 등 사용자 UI 디자인 변경|
|Comment|필요한 주석 추가 및 변경|
|Rename|파일 또는 폴더명을 수정하거나 옮기는 작업만의 경우|
|Remove|파일을 삭제하는 작업만 수행한 경우|
|!HOTFIX|급하게 치명적인 버그를 고쳐야 하는 경우|
|Modify|코드 단순 수정|
